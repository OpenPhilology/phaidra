Plural
1st ἄρξομεν We will command

Future Indicative Active

There are some exceptions to the way in which the future is formed. Some verbs end in consonants that, when meeting σ, change into different consonants.

  • κ / γ / χ + σ changes into ξ
  • π / β / φ + σ changes into ψ
  • τ / δ / θ + σ changes into σ
